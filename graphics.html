ggplot2

  • input data (data.frame)
  • aes - color, format and size
  • geoms - geometric objects (points, lines, …)
  • facets - forms a matrix of panels defined by row and column faceting variable
  • stats - statistical transformations
  • scales

ggplot2 - basic object

  • ggplot(data, aes(x,y))
  • geom_bar()
  • geom_point()
  • geom_line()

ggplot2 - adding metadata

  • ggtitle()
  • xlab()
  • ylab()

ggplot2 - example

  • line chart
g<-ggplot(index,aes(x=year,y=pop, group=1))+geom_line()
g<-g+ggtitle("Population by year")+xlab("Year")+ylab("Population")
g

ggplot2 - additional options in aes part

  • fill: colour of data in a barchar
ggplot(data=mtcars, aes(x=as.factor(cyl), fill=factor(gear))) + 
  geom_bar()+xlab("Num. of cyl")+ylab("")+labs(fill="Num. of gears")

ggplot2 - setting axis limits+changing stat function

  • we already have data aggregated, so we have to use stat="identity"
  • xlim, ylim specify the range of scale
ggplot(index,aes(x=as.factor(year), y=pop))+geom_bar(stat="identity")+ 
  coord_cartesian(ylim=c(10000,20000))+ylab("Population")+xlab("year")

ggplot2 - geom_smooth()

  • aids the eye in seeing patterns in the presence of
  • method: lm, glm,…
