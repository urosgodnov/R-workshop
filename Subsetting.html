Atomic vectors - positive integers

x <- c(2.1, 4.2, 3.3, 5.4)
x[c(3, 1)]
## [1] 3.3 2.1
x[order(x)]
## [1] 2.1 3.3 4.2 5.4
x[c(1, 1)]
## [1] 2.1 2.1
x[c(2.1, 2.9)] # Real numbers are silently truncated to integers
## [1] 4.2 4.2

Atomic vectors - negative integers

  • omit elements at the specified positions
  • you can’t mix positive and negative integers in a single subset
x <- c(2.1, 4.2, 3.3, 5.4)
x[-c(3, 1)]
## [1] 4.2 5.4

Atomic vectors - logical vectors

x <- c(2.1, 4.2, 3.3, 5.4)
x[c(TRUE, TRUE, FALSE, FALSE)]
## [1] 2.1 4.2
x[x > 3]
## [1] 4.2 3.3 5.4

List

  • subsetting a list works in the same way as subsetting an atomic vector
  • using [ will always return a list; [[ and $ let you pull out the components of the list
mylist<-list(x=c(1,5,7), y=c(4,2,6), z=c(0,3,4))
mylist
## $x
## [1] 1 5 7
## 
## $y
## [1] 4 2 6
## 
## $z
## [1] 0 3 4

List

mylist[[1]] #first element
## [1] 1 5 7
mylist[[2]][[3]] #third element of a second element
## [1] 6
mylist$z[[2]] #by name - second element of a third element
## [1] 3

Matrices and arrays

a <- matrix(1:9, nrow = 3)
colnames(a) <- c("A", "B", "C")
a[1:2, ] ## first 2 rows and all columns
##      A B C
## [1,] 1 4 7
## [2,] 2 5 8
a[2:3,2:3] ## second and third row and second and third column
##      B C
## [1,] 5 8
## [2,] 6 9

More advanced subsetting

  • colnames() returns the name of the function
  • grepl() returns true, where the condition is found
# column names
colnames(airquality)
## [1] "Ozone"   "Solar.R" "Wind"    "Temp"    "Month"   "Day"
# column names with the letter "n"
grepl("n",colnames(airquality))
## [1]  TRUE FALSE  TRUE FALSE  TRUE FALSE
#returning only columns from a dataframe that have "n" in name
head(airquality[,grepl("n",colnames(airquality))],3)
##   Ozone Wind Month
## 1    41  7.4     5
## 2    36  8.0     5
## 3    12 12.6     5

More advanced subsetting

  • returning every other column
  • use seq() function
# number of columns
ncol(airquality)
## [1] 6
#create sequence
seq(1,ncol(airquality),by=2)
## [1] 1 3 5
#returning only columns from a dataframe that have "n" in name
head(airquality[,seq(1,ncol(airquality),by=2)],3)
##   Ozone Wind Month
## 1    41  7.4     5
## 2    36  8.0     5
## 3    12 12.6     5

Subsetting with the subset() function

  • subset(x, subset, select)
  • x: dataframe, subset: filter rows, select: columns
head(subset(airquality,airquality$Temp>75 & airquality$Wind>12, c("Temp","Month")), 3)
##    Temp Month
## 29   81     5
## 37   79     6
## 40   90     6
